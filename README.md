# Library Management System

A simple Library Management System developed using Flask and SQLite.

## Features

- Login Page
- Dashboard
- Book Management
- Add Books
- Delete Books
- SQLite Database
- Responsive Bootstrap UI

## Software Requirements

Python 3.10+

Flask

SQLite

VS Code

## Installation

Install Flask

```bash
pip install -r requirements.txt
```

Create Database

```bash
python database.py
```

Run Project

```bash
python app.py
```

Open Browser

```
http://127.0.0.1:5000
```
